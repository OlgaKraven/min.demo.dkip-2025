# Модуль 2. Разработка базы данных

## Задание

Создайте базу данных на основании разработанной ER-диаграммы, используя предпочтительную платформу. Создайте таблицы основных сущностей, атрибуты, отношения и необходимые ограничения. После создания базы данных требуется импортировать данные из файла `Заказчики.json`.
