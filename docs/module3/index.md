# Модуль 3. Создание запроса

## Задание

Создайте запрос, позволяющий вычислить **полную стоимость заказа покупателя** с учётом:

- количества продукции в заказе;
- стоимости всех материалов, использованных для производства данной продукции (учитывая норму расхода).
