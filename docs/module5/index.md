# Модуль 5. Разработка проектной документации

## Задание

Разработайте проектную документацию на разработанный функционал. Включите описание функционального назначения, используемые методы с указанием параметров.

## Приложения

Необходимые приложения: **отсутствуют**.

## Состав документации

В рамках задания рекомендуется подготовить:

1. **Пояснительная записка** — общее описание проекта, архитектура, используемые технологии, описание классов и методов.
2. **Техническое задание** — требования к системе по ГОСТ 19.201–78.
3. **Пользовательская документация** — руководство по эксплуатации по ГОСТ 19.505–79.
