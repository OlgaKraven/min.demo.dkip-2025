# Подготовка к демоэкзамену ДКИП — Базовый уровень

> Это **не эталонный** пример для подготовки к демоэкзамену.
> Весь проект рассмотрен на основе официального задания по примеру экзамена 2026 года.

---

## Параметры экзамена

| Параметр | Значение |
|----------|---------|
| Специальность | 09.02.07 Информационные системы и программирование |
| Уровень | **Базовый (ДЭ БУ)** |
| Продолжительность | **2 часа 30 минут** |
| Проходной балл | **40** из 75 |
| Официальная документация | [bom.firpo.ru/Public/5510](https://bom.firpo.ru/Public/5510) |

---

## Модули базового уровня

| № | Модуль | Что делать |
|:-:|--------|-----------|
| 1 | Проектирование ER-диаграммы | Спроектировать ER-диаграмму (3НФ, PK/FK), сохранить в PDF |
| 2 | Разработка базы данных | Создать БД по ER-диаграмме, импортировать `Заказчики.json` |
| 3 | Создание запроса | Написать запрос расчёта полной стоимости заказа покупателя |
| 4 | Разработка ИС | Форма авторизации: логин, пароль, CAPTCHA-пазл, блокировка (3 попытки), панель администратора |
| 5 | Проектная документация | Пояснительная записка, ТЗ, пользовательская документация |
