# Модуль 4. Разработка информационной системы

## Задание

Разработайте форму авторизации зарегистрированных пользователей с ролями **«Администратор»** и **«Пользователь»**.

### Требования к форме авторизации

- Текстовые поля: **Логин** и **Пароль** (оба обязательны для заполнения).
- Кнопка **«Войти»**.
- При неверных данных: `"Вы ввели неверный логин или пароль. Пожалуйста проверьте ещё раз введенные данные"`.
- После успешной авторизации: `"Вы успешно авторизовались"`.
- Интерактивная **капча-пазл**: пользователь собирает изображение из фрагментов (метод произвольный). Система проверяет правильность расположения фрагментов.

### Блокировка

Если **3 раза подряд** пазл собран неверно или пароль введён неверно — учётная запись блокируется. При попытке входа заблокированного пользователя: `"Вы заблокированы. Обратитесь к администратору"`.

### Администратор

Рабочий стол администратора должен содержать:
- добавление новых пользователей (с проверкой уникальности логина);
- изменение данных текущих пользователей;
- снятие блокировки.
